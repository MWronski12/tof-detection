void tmf882x_close ( struct tmf882x_tof *  tof)

Close the core driver firmware interface, the inverse operation of tmf882x_open().

Device will be put into STANDBY power state. tmf882x_open() must be called again to perform any operations on the device.

Parameters
[in]toftof dcb interface context
Returns
0 for sucess, otherwise failure
